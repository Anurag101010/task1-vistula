# Task 1 - Spring Boot Application

## Project Description
Simple Spring Boot application demonstrating MVC pattern with two endpoints.

## Endpoints
1. `GET /hello` - Returns plain text response with `@ResponseBody`
2. `GET /greeting` - Returns HTML view with image (Thymeleaf template)
